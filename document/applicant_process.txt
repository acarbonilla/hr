How to prevent applicant if failed under 1  month?
Follow up question: Is it manually or automatic if failed and date to re-applied.


Steps:

1. Landing Page: Applicant visits / and clicks "Get Started".
2. Position Selection: Applicant is redirected to /positions and selects a position (e.g., "Virtual Assistant").
3. Sign Up / Sign In:
    If new, applicant completes registration at /register?position=virtual-assistant.
    If returning, applicant logs in at /login?position=virtual-assistant.
4. AI Interview: Applicant is redirected to /interview/[id] and completes video interview.
5. Transcribing: System transcribes the video response.
    Start timer when transcription begins.
    End timer when transcription completes.
6. Analyzing: System analyzes the transcribed text with AI.
    Start timer when analysis begins.
    End timer when analysis completes.
7. Result: Applicant receives feedback/results.
