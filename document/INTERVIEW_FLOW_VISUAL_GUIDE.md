### 1. Initial Countdown (5 seconds)

- **Visual**: Full-screen purple-blue gradient
- **Elements**: Large countdown number, info cards
- **Audio**: Silent (preparing for first question)
- **Purpose**: Mental preparation, reduce anxiety

### 2. Question Display

- **Visual**: Clean white card with question text
- **Elements**:
  - Question number (e.g., "Question 1 of 5")
  - Question type badge (Behavioral, Technical, etc.)
  - Speaking indicator (green, animated)
  - Replay button (blue, top-right)
  - Tips section with emojis
- **Audio**: Auto-plays question via text-to-speech
- **Interaction**: Click replay button to hear again

### 3. Recording Phase

- **Visual**: Live webcam feed, recording timer
- **Elements**:
  - Red "Recording" indicator
  - Time counter (MM:SS)
  - Stop button
- **Audio**: Voice stops during recording
- **State**: User is actively answering

### 4. Processing Phase

- **Visual**: Modal overlay with loading animation
- **Elements**:
  - Spinning loader
  - Progress indicators (3 steps)
  - Motivational message
- **Audio**: Silent
- **Duration**: 30-60 seconds (AI analysis)

### 5. Transition Phase

- **Visual**: Full-screen green-blue gradient
- **Elements**:
  - Large countdown (3-2-1)
  - "Great Job!" celebration
  - Pulsing circle animation
- **Audio**: Silent (preparing for next question)
- **Duration**: 3 seconds

### 6. Auto-Advance

- **Action**: Automatically moves to next question
- **Visual**: Smooth fade transition
- **Audio**: New question reads immediately
- **User Action**: None required (automatic)
